# SGF-WU – Siber Güvenlik Farkındalık Web Uygulaması

SGF-WU, Beykent Üniversitesi Yazılım Mühendisliği Bölümü bitirme projesi kapsamında geliştirilen web tabanlı bir siber güvenlik farkındalık uygulamasıdır.

## 📌 Proje Hakkında

Uygulama, kullanıcıların parola güvenliği ve güvenli internet kullanım alışkanlıklarını analiz ederek siber güvenlik farkındalık düzeylerini değerlendirmektedir. Ayrıca kişiselleştirilmiş geri bildirimler ve demografik analizler sunmaktadır.

## 🚀 Özellikler

- 🔐 Parola güvenliği analizi
- 🧠 Siber güvenlik farkındalık testi
- 📊 Toplam farkındalık skoru hesaplama
- 💡 Kişiselleştirilmiş geri bildirim ve öneriler
- 👨‍💼 Yönetici paneli
- 📈 İstatistiksel analizler
- 📋 Demografik analizler
- 📉 Grafik tabanlı raporlama

## 🛠️ Kullanılan Teknolojiler

- Python
- Streamlit
- SQLite
- SHA-256
- Pandas
- Matplotlib

## 🎯 Projenin Amacı

Kullanıcıların parola güvenliği ve internet kullanım alışkanlıklarını değerlendirerek siber güvenlik farkındalıklarını ölçmek ve geliştirmek; kişiselleştirilmiş geri bildirimler ile daha güvenli dijital davranışların desteklenmesini sağlamaktır.
